# Human Activity Recognition Using Hidden Markov Models

Formative 2 — Machine Learning Techniques

## Overview

This project builds a Hidden Markov Model from scratch to classify four physical activities — standing, walking, jumping, and still — using real smartphone sensor data collected with Sensor Logger on an iPhone 13 Pro.

## Results

Overall cross-validation accuracy: **93.8%**

| Activity | Sensitivity | Specificity |
|---|---|---|
| standing | 0.947 | 0.947 |
| walking | 0.947 | 0.976 |
| jumping | 0.993 | 0.996 |
| still | 0.868 | 1.000 |
